   14: =head1 NAME
   15: 
   16: Text::Names::Abbreviate - Create abbreviated name formats from full names
   17: 
   18: =head1 SYNOPSIS
   19: 
   20:   use Text::Names::Abbreviate qw(abbreviate);
   21: 
   22:   say abbreviate("John Quincy Adams");		 # "J. Q. Adams"
   23:   say abbreviate("Adams, John Quincy");		# "J. Q. Adams"
   24:   say abbreviate("George R R Martin", { format => 'initials' }); # "G.R.R.M."
   25: 
   26: =head1 DESCRIPTION
   27: 
   28: This module provides simple abbreviation logic for full personal names,
   29: with multiple formatting options and styles.
   30: 
   31: The input is expected to be a personal name consisting of one or more
   32: whitespace-separated components. These are typically interpreted as:
   33: 
   34:   First [Middle ...] Last
   35: 
   36: Names consisting of a single component are treated as a single name,
   37: and no abbreviation of given names is possible.
   38: 
   39: =head1 SUBROUTINES/METHODS
   40: 
   41: =head2 abbreviate
   42: 
   43: Make the abbreviation.
   44: It takes the following optional arguments:
   45: 
   46: =over
   47: 
   48: =item format
   49: 
   50: One of: default, initials, compact, shortlast
   51: 
   52: C<Shortlast> produces initials followed by the full last name, ignoring style reordering.
   53: This is similar to the default format but does not support C<last_first> output.
   54: 
   55: =item style
   56: 
   57: One of: first_last, last_first
   58: 
   59: =item separator
   60: 
   61: String used between initials (default: ".").
   62: 
   63: Note that spacing between initials is handled separately depending on
   64: the selected format.
   65: 
   66: =back
   67: 
   68: =head3 Name Formats
   69: 
   70: The function recognizes names in both of the following forms:
   71: 
   72: =over 4
   73: 
   74: =item *
   75: 
   76: C<First Middle Last>
   77: 
   78: =item *
   79: 
   80: C<Last, First Middle>
   81: 
   82: =back
   83: 
   84: In the latter case, the name will be normalized internally before
   85: abbreviation.
   86: 
   87: If the input begins with a comma (e.g., C<", John">), it is interpreted
   88: as having no last name, and only initials will be produced.
   89: 
   90: =head3 Errors
   91: 
   92: The function will throw an exception (via C<croak>) if:
   93: 
   94: =over 4
   95: 
   96: =item *
   97: 
   98: The C<name> parameter is missing or undefined
   99: 
  100: =item *
  101: 
  102: The C<name> parameter is an empty string
  103: 
  104: =item *
  105: 
  106: An invalid value is provided for C<format> or C<style>
  107: 
  108: =back
  109: 
  110: =head1 EXAMPLES
  111: 
  112:   abbreviate("Madonna")
  113:   # "Madonna"
  114: 
  115:   abbreviate("Adams, John Quincy")
  116:   # "J. Q. Adams"
  117: 
  118:   abbreviate("John Quincy Adams", { style => 'last_first' })
  119:   # "Adams, J. Q."
  120: 
  121:   abbreviate("John Quincy Adams", { format => 'compact' })
  122:   # "JQA"
  123: 
  124: =head3 Notes
  125: 
  126: Abbreviation formats such as C<compact> and C<initials> are lossy
  127: transformations. They discard structural information about the original
  128: name.
  129: 
  130: As a result, passing the output of C<abbreviate()> back into the function
  131: may not yield equivalent results:
  132: 
  133:   abbreviate("George R R Martin", { format => 'compact' })   # "GRRM"
  134:   abbreviate("GRRM", { format => 'initials' })               # "G."
  135: 
  136: In such cases, the input is treated as a single name.
  137: 
  138: Initials are derived by taking the first character of each name component verbatim.
  139: No filtering is applied,
  140: so non-alphabetic characters (such as punctuation or digits) will be included as-is.
  141:

  143:

  145: 
  146:   {
  147:     'name' => { 'type' => 'string', 'min' => 1, 'optional' => 0 },
  148:     'format' => {
  149:       'type' => 'string',
  150:       'memberof' => [ 'default', 'initials', 'compact', 'shortlast' ],
  151:       'optional' => 1
  152:     }, 'style' => {
  153:       'type' => 'string',
  154:       'memberof' => [ 'first_last', 'last_first' ],
  155:       'optional' => 1
  156:     }, 'separator' => {
  157:       'type' => 'string',
  158:       'optional' => 1
  159:     }
  160:   }
  161: 
  162: =head4	OUTPUT
  163: 
  164: Argument error: croak

  166:   {
  167:     'type' => 'string',
  168:   }
